yac
===

Yet another chat...

## Libraries

* Angular.js 1.3
* Firebase 1.0
* Bootstrap 3.1
* LoDash 2.4

## Status of the app (working)

* Login / logout
* store user in session
* stores messages
* message markdown
* show sender avatar
* FB security roles
* Delete messages (own only)

### Todo
