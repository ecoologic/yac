yac
===

Yet another chat...

## Libraries

* Angular.js 1.3
* Firebase 1.0
* Bootstrap 3.1
* LoDash 2.4

## Working features

* Login / logout
* Store user in session
* Some Firebase security roles
* Show user avatar
* Stores messages
* Delete messages (own only)
* Message markdown
* Icons in messages
